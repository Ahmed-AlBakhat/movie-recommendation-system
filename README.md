# AI Movie Recommendation System

## Overview
This project is a simple AI-based movie recommendation system that suggests movies based on the user's preferences. The system combines multiple recommendation techniques to provide personalized movie suggestions.

## AI Methods Used
- Content-Based Filtering
- Item-Based K-Nearest Neighbors (KNN)
- Singular Value Decomposition (SVD)

## Tools Used
- Python 3.13
- NumPy
- Scikit-learn

## Project Files
- recomendation system.py
- README.md

## Features
- Select favorite movie genres.
- Rate movies from 1 to 5.
- Personalized movie recommendations.
- Hybrid recommendation model.
- Cold-start support for new users.

## Results
The system successfully:

- Generated personalized movie recommendations.
- Recommended unseen movies based on user preferences.
- Combined Content-Based Filtering, Item-Based KNN, and SVD to improve recommendation accuracy.
